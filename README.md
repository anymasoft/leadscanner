# LeadScanner

**Фоновый сканер источников лидов с планировщиком и REST API.**

Сервис по расписанию опрашивает источники, выявляет потенциальных клиентов и отдаёт результат через API. Лёгкая альтернатива тяжёлым CRM-парсерам для непрерывного мониторинга.

---

## Возможности

- **Планировщик** (APScheduler) — регулярный опрос источников без ручного запуска.
- **REST API** (FastAPI) для управления заданиями и выдачи результатов.
- **Асинхронная обработка** — неблокирующий сбор данных.

---

## Стек

FastAPI · APScheduler · asyncio

## Быстрый старт

```bash
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python main.py
```
